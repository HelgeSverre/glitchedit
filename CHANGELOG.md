# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2024-12-30

### Added
- npm distribution (`npx glitchedit`)
- Cross-platform standalone binaries (Linux, macOS, Windows)
- GitHub Actions CI/CD workflows
- Version display in header with GitHub link
- CHANGELOG.md

### Changed
- Updated README with installation options and badges

## [1.0.0] - 2024-12-30

### Added
- PNG glitch art editor with hex editing capabilities
- Visual effects system with 20+ built-in effects
- Real-time preview of changes
- Chunk-aware navigation for PNG structure
- Keyboard shortcuts for power users
- Single-file HTML bundle for easy sharing
- CLI tool for local development (`npx glitchedit`)
- Cross-platform standalone executables
- Help system with effect documentation and previews

### Features
- **Hex Editor**: Full-featured hex editor with ASCII view
- **Effects Panel**: Apply and stack multiple glitch effects
- **Chunk Navigator**: Browse PNG chunks (IHDR, IDAT, etc.)
- **Live Preview**: See changes in real-time
- **Random Image**: Generate random test images
- **Export**: Download modified PNG files
